# 📲 First Message Templates

> بتتبعت أوتوماتيك بعد ملء الفورم (max 2 دقيقة)
> Variables: `{{name}}` و `{{field}}`

---

## Template A — Generic

```
أهلاً {{name}}!
أنا أحمد من Adstartup — بندير متاجر Amazon وNoon وJumia.
بتبيع دلوقتي online وﻻ لسه في البداية؟
```

## Template B — Fashion

```
أهلاً {{name}}!
الملابس دلوقتي من أكتر الفئات على Noon وJumia في مصر.
عندك ستوك جاهز وﻻ لسه بتجهز؟
```

## Template C — Electronics

```
أهلاً {{name}}!
الإلكترونيات على Amazon Egypt شغالة بقوة.
عندك منتجات جاهزة وﻻ لسه في مرحلة الاستيراد؟
```

## Template D — Home

```
أهلاً {{name}}!
منتجات البيت على Amazon وRaneen بيطلبوا عليهم كتير.
بتصنع وﻻ بتستورد؟ وعندك ستوك دلوقتي؟
```

---

## Follow-up — بعد 24 ساعة

```
أهلاً {{name}}، أنا أحمد من Adstartup.
لو الوقت مش مناسب مفيش مشكلة —
بس لو عندك 5 دقايق هفيدك بمعلومة مهمة.
```

## Final — بعد 3 أيام

```
أهلاً {{name}}،
مش عايز أزعجك. لو قررت تشتغل على المنصات في أي وقت، أنا هنا. 🙂
```

---

## ⏰ Timing Rules

| الحالة | Action |
|--------|--------|
| ملأ الفورم | فوراً (max 2 دقيقة) |
| مردش نفس اليوم | Follow-up بعد 24 ساعة |
| مردش بعد 3 أيام | Final follow-up |
| "مش دلوقتي" | Reschedule بعد 2 أسبوع |

## 📌 Rules

1. مش أكتر من 3 جمل
2. دايماً ينتهي بسؤال واحد
3. باسم العميل دايماً
4. مفيش "يسعدني" أو "بكل سرور"
